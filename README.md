# task-0.3

📌 Project Title

Web Scraper for News Headlines

📖 Description

This project is a simple Python-based web scraper that extracts top news headlines from a public news website and saves them into a text file. It demonstrates the basics of web scraping using requests and BeautifulSoup.

🚀 Features
Fetches live news headlines
Parses HTML content
Removes duplicate headlines
Saves output to .txt file
Easy to run and modify
🛠️ Tech Stack
Python
requests
BeautifulSoup
📂 Project Structure
📁 News-Web-Scraper
│── scraper.py
│── news_headlines.txt
│── README.md
⚙️ Installation
Step 1: Clone the repository
git clone https://github.com/your-username/news-web-scraper.git
cd news-web-scraper
Step 2: Install dependencies
pip install requests beautifulsoup4
▶️ Usage

Run the script:

python scraper.py
📄 Output
Extracted headlines will be displayed in the console
Saved in:
news_headlines.txt
